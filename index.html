Rancher Cluster Templates Helm Charts

Usage

helm repo add rancher-cluster-templates https://bashofmann.github.io/rancher-cluster-templates

This repository contains RKE2 Cluster template Helm charts that are used for demonstration purposes.

The Helm charts in this repository are not production ready and are meant for demonstration purposes only!
